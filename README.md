# 🌍 Travline – Travel Landing Page

A modern, responsive **travel landing page website** designed to showcase destinations, services, and travel experiences. This project focuses on clean UI, smooth layout, and engaging sections for travel businesses or portfolios.

---

## 🚀 Live Demo

👉 https://travline-7pfr.vercel.app/

---

## ✨ Features

* 🌄 Beautiful hero section with CTA
* 🧭 Travel destinations showcase
* 🛎️ Services section (Tour Guide, Easy Booking, etc.)
* 🏝️ Featured destinations with pricing & ratings
* 📩 Newsletter subscription UI
* 📱 Fully responsive design
* 🎨 Clean and modern UI/UX

## ⚙️ Installation & Setup

Since this is a static website, no build tools or dependencies are required.

### Option 1: Run Locally (Simple Way)

1. Clone the repository:

```
git clone https://github.com/your-username/travline.git
cd travline
```

2. Open the project:

* Simply open `index.html` in your browser

---

### Option 2: Using Live Server (Recommended)

1. Install **Live Server** extension in Visual Studio Code
2. Right-click on `index.html`
3. Click **"Open with Live Server"**
